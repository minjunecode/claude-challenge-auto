1시간마다 자동 실행 등록
PC가 켜져 있으면 매 시간 자동으로 사용량을 보고하도록 등록합니다.
노트북 배터리 모드에서도 정상 작동합니다.
a
PowerShell 열기:
키보드에서 Win 키를 누르고 "powershell" 이라고 입력하세요.
b
검색 결과에서 "Windows PowerShell"을 찾아 "관리자 권한으로 실행"을 클릭하세요.
(우클릭하면 나오거나, 오른쪽에 바로 보입니다)
c
"이 앱이 변경을 허용하시겠습니까?" 라고 물으면 "예"를 클릭하세요.
파란 배경의 창이 열립니다. 이게 PowerShell입니다.
d
아래 코드 블록에서 첫 번째 줄만 수정하면 됩니다.

① 아래 노란색 글씨를 클릭하세요 — 입력창이 뜹니다
② Step 2에서 복사한 내 Python 경로를 붙여넣고 확인
③ 경로가 초록색으로 바뀌면 "복사" 버튼을 누르세요
# 첫 줄의 경로만 내 것으로 바꾸세요 (클릭!)

$py = "여기를_클릭해서_내_Python_경로_입력"

# 아래는 수정하지 마세요 ─────────────
$action = New-ScheduledTaskAction -Execute $py -Argument "$env:USERPROFILE\.claude\challenge-report.py"
$trigger = New-ScheduledTaskTrigger -Once -At "00:00" -RepetitionInterval (New-TimeSpan -Hours 1)
$settings = New-ScheduledTaskSettingsSet -AllowStartIfOnBatteries -DontStopIfGoingOnBatteries -StartWhenAvailable -ExecutionTimeLimit (New-TimeSpan -Minutes 10)
Register-ScheduledTask -TaskName "ClaudeChallenge" -Action $action -Trigger $trigger -Settings $settings -Force
완성 예시 (첫 줄만 다릅니다):
$py = "C:\Users\홍길동\AppData\Local\Programs\Python\Python312\python.exe"
e
PowerShell 창(파란 화면)에서 마우스 우클릭하면 자동으로 붙여넣기가 됩니다.
(Ctrl+V는 안 될 수 있습니다. 반드시 우클릭으로 붙여넣기)
f
붙여넣기가 되었으면 Enter 를 누르세요.
g
아래처럼 나오면 성공입니다! PowerShell 창을 닫아도 됩니다.
TaskPath TaskName State

확인 방법: 챌린지 사이트 → 인증 탭에서 "자동 리포팅 활성" 표시가 나오면 성공!

Win → "powershell" 검색 → 관리자 권한으로 실행 → 아래 입력 후 Enter
Unregister-ScheduledTask -TaskName "ClaudeChallenge" -Confirm:$false
