토큰 찾는 법 (Claude Code 설치 필요)
1
터미널(또는 명령 프롬프트)을 열고 아래 명령어를 입력하세요:

Windows:
type %USERPROFILE%\.claude\.credentials.json

Mac/Linux:
cat ~/.claude/.credentials.json
2
아래와 같은 내용이 출력됩니다:

{"claudeAiOauth":{"accessToken":"sk-ant-oat01-abcdef...여기가_토큰","refreshToken":"sk-ant-ort01-...","expiresAt":...}}
3
sk-ant-oat01-로 시작하는 부분만 복사하세요.
"accessToken":" 뒤부터 다음 " 앞까지의 값입니다.

* refreshToken, expiresAt 등 다른 값은 복사하지 마세요
4
복사한 토큰을 위의 입력창에 붙여넣기 후 등록 버튼을 누르세요.
